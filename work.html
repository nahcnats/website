ClockIT

This is a mobile time attendance system. Developed using VueJs for the admin frontend, React-Native for the mobile front-end, NodeJS / ExpressJs for the business logic at the server and MySql as the database.

Development started in mid 2018 and was released to market end of 2019. The mobile application is both available in Apple Store and Google Playstore. Click the YouTube below for a sample of how it works.
